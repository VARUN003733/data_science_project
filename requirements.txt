# Data Science Pro

This project is a data science application built with Flask and NumPy.

## Requirements

- Python 3.8+
- Flask >= 3.1.2
- NumPy >= 2.3.4

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd data_science_pro
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```
   python app.py
   ```

2. Open your browser and go to `http://localhost:5000`.

## Project Structure

- `app.py` - Main application file.
- `requirements.txt` - Python dependencies.
- Other scripts and notebooks for data processing and analysis.
